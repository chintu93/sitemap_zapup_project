Wrike is a project management tool where you can plan and organize your projects, tasks, and folders. You can customize your dashboard, synchronize tasks and milestones with calendar, and be up to date with the live activity stream.

How Does it Work

Zapup automates your work by performing Actions (like “Create Task”) to complete an action easily in Wrike while a trigger occurs in another application. You can easily create a folder, task or add a comment using this integration.

ACTIONS

Create project

Creates a new project

Create task

Creates a task linked to a project

Add comment

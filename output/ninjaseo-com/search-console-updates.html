What is Google Search Console?

Google Search Console by Google helps in monitoring, maintaining and troubleshooting your sites’ Google search results. It is a free service and does not require any registration but it can be of great benefit in helping you understand how Google views your site.

Search Console offers reports and tools that help you:

  • Confirm Googles ability to find your site and crawl it.
  • Fix issues in indexing and request re-indexing to include content that is new or updated.
  • View the traffic data on Google Search to determine whether your site appears often on google, what search queries prompt it and how often searchers are directed to it.
  • Receive alerts every time Google encounters spam, indexing, or other issues relating to your site.
  • Give you a view of the sites that link to your website.
  • Troubleshoot mobile usability, AMP and other Search features issues.
  • Every once in a while, Google will add new features to better help webmasters in their analysis and the improvement of their websites’ performance.

Google Search Console Updates in August 2020

Google AMP Now Supports Signed Exchange

Google AMP is now supporting a signed exchange that will allow publishers to convey portable content safely without losing integrity.

August 2020 –

Google Search Console API Upgrade

A recent upgrade in the Google search console was initiated and has added features to enhance backward functionality. Other enhancements included in this upgrade changes on Google cloud platform as well as changes in the API key restriction.

Image License Documentation Update

Google updated their Image License documentation which is now out of beta. All images have to be formatted ad referenced in formats that are supported by Google Images. The images on your website can now display with a Licensable badge on image thumbnails in Google Images. This also provides a link to the license in the Image Viewer.

Google Search Console Updates in July 2020

Product Rich Results Update

Google updated product rich results to support pages on a single product as opposed to category pages or lists. Also implemented Lighthouse recommendations to specify what small font size means on the Mobile SEO guidelines. Google also specified that logos can be in any format supported by Google Images. They also added features to monitor rich results that provide structured data such as recipes and FAQs

Google Search Console Updates in June 2020

Lazy Loading Guide Update

Google updated the lazy-loading guide which defers loading of non-critical or non-visible content. If this is not done well it can prevent some pages from showing up on Google search results. The update ensures that uploaded content is visible in the viewport.

Google Search Console Updates in May 2020

Page Experience Update

As from May, page experience will show up Google Search results. This enhanced features will enhance Web stories on Google. Page experience will serve as a signal to guide how a page is ranked on Google search results. However, this will not beat the quality of content on the page. Google also updated guidance on reducing the crawl rate on Googlebot. Google also the discovery of social profiles and corporate contact information on Google knowledge panels

Google Search Console Updates in April 2020

Debugging Policies Update

Google updated policies on debugging specifying that cached links are not a reliable way to achieve this. The recommended tool for this purpose is the URL Inspection Tool which is up-to-date which can be used for general debugging, JavaScript debugging, and structured data debugging. Early in April, Google also updated requirements for submitting COVID-19 announcements on the search console. This is restricted to national government domains and state level agencies.

Google Search Console Updates in March 2020

Pausing an Online Business Update

Google updated requirements for pausing your online business. This feature allows business owners to temporarily close their online business in case they are out of stock or products. Google will allow business owners to disable certain features such as cart functionality and let Google know about this changes. Disabling the whole website is however not recommended.

Google Search Console Updates in February 2020

Google made major updates in February that touch on many areas as elaborated below:

February 26: Better Data Export in Search Console

The Search Console team made more data that can be exported available to users on the Google Search Console. This data can be easily downloaded which includes all the data from Search Console Reports. This will make the data easier to read and store as opposed to the previous specific table views. With this, the full reports will be accessible for a long duration.

Google Search Console Update - Enhancement Reports

Google also made updates on their enhancement reports. This makes it possible to export data from reports behind the charts. This reports give a clear breakdown of data such as page impressions, Google status and other significant issues that may be affecting the pages. This is critical data for website management.

Google Search Console Update- Performance Report

Google has also given updates for performance reports. These reports can be easily downloaded in the form of Excel spreadsheet or CSV files that can be compressed in a zip file. All this is done using different tabs. Together with the performance data using a filters tab the user can extract all the data filters applied in the exportation of the data.

Additional Ways to use Search Console Data

There are different ways in which a user can use search console data. With a technical background the user may consider the Search Console API which can be used to add, view or remove properties and sitemaps. This can also be used run advanced queries on Google to generate Search results data.

The Google Data Studio offers a dashboarding solution that is easy to implement. Users can aggregate data from multiple sources. The Search Console connector allows users to import different metrics and dimensions to the user dashboard for side-by-side comparison with data from other tools.

Change of Address Features – 19-02-2020

In February 2020, Google made updates to the site address within the new search console. This update is within the search settings of each web property and comes with a few additional features. This includes changes within the change of address section to show whether the address change has been successful or otherwise. This change is critical when a website decides to alter its domain name or shift the subdomain to a different location.

Once the change of address has been initiated using the appropriate tool, Google will make it a priority to index the new address of the site and ignore the old address. This should happen gradually over time.

Once the redirects have been instituted, this should be the 301 Redirect, the webmasters will use the change of address tool located in the search console to notify Google Crawlers of the shift to the new domain or subdomain.

Google Search Console Update- Redirect Validation

As from Feb 2020, Google will start to update a few new features targeted at helping the webmasters validate redirect requests. This is primarily a complete change of prerequisite 301 redirects.

The search engine crawlers will be redirected to the new domain with all the authority and trust gained by the old domain which will be shifted to the new one. In addition this will allow all users trying to access the former domain to be redirected to the new site automatically.

With the former validation feature, it was not possible for webmasters to tell whether redirects were successful. With this update on Redirect Validation that has been added to the search console, webmasters will get a clear picture on the top 5 URLs of the domain. This means having a successful 301 redirect.

Google Search Console Update - On-screen Reminder

Even with the update on the Validation Request, it still takes Google a couple of months to redirect the entire ranking signals from an old website to a new one. Approximately a successful redirect should take about 180 days for the whole process to go through and all the signals to be passed. This also varies based on the number of pages on the site and the number of backlinks that have been generated. With this recent Google update webmasters will begin to get a reminder or notification on the console dashboard regarding the process. This may be a signal such as “The site is currently migrating to www.xyz.com.” This should last for the entire migration period.

However, this feature is restricted for those who are new to SEO. This means that this 301 redirect tool may not be implemented on such a website. In this case, it has to be implemented as part of the Google Search Console directly within the website before the Change of Address feature can be used.

Google Updated Their Review Snippet Policy

Google announced that it will be giving more emphasis on review snippets. This have been around for some time but had been pushed to the back burner. Google declared that this tool has which is a part of structured data types has been greatly misused. Consequently, Google was going to mitigate this misuse by making it available on a select webpage types such as reviews of movies and products as well as Software.

The review snippet functions as a Clink-Through magnet due to star rating and other similar features. With this update webmasters will be able to troubleshoot the markup implementation. As announced by Google, they will also be able to review the markup code prior to going live. However, this feature will only be available to sites that have implemented the review markup on the console dashboard. This will allow users to view warnings, notifications, and errors, for markup that have been implemented on the site.

Additionally, this enhancement feature on the review snippet will show up on the Performance Report filter to help the webmasters to have a better understanding on the performance of the review snippet. This will be done using filters.

Google Search Console Update in January 2020

Removals Report Update

This update was made at the start of the year and was added to the Google Search Console to stop the search engine from displaying your site temporarily. With this, webmasters can temporarily remove a specific page from appearing on Google search. This will also allow them to get information relating to takedown requests that have been filed using Google public tools. Lastly, they will also be able to temporarily bring down URLs that have errors or contention.

Google has implemented and updated three types of tools to undertake this removal function.
